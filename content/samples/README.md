# Sample PlayerCharacter Definitions

This folder contains example `PlayerCharacter` builds that can be fed into
`CharacterCreator.build_character`. Each JSON file includes the creation
selection plus a small `expectations` block that documents derived stats for
quick sanity checks.

- **human_ranger_3.json** – Human Ranger (level 3) built from the standard
  array. Expected 36 HP, AC 14, trained in Insight, Persuasion, Stealth, and
  Survival, with the Keen Senses and Warding Light feats.
- **dusk_elf_cleric_3.json** – Dusk Elf Battle Cleric (level 3) using the
  standard array. Expected 27 HP, AC 13, trained in Insight, Persuasion,
  Stealth, and Survival, also with Keen Senses and Warding Light.
